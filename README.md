# netrum-auto
automatic mining, claim every 24h send notif telegram

1. Clone
   ```
   git clone https://github.com/rokipdj88/netrum-auto.git
   ```
   
2. Config .env
   Change BOT_TOKEN, CHAT_ID, WALLET
   
   ```
   nano .env
   ```
   
3. Permission
   ```
   chmod +x netrum-auto.sh

4. Create Screen
   ```
   screen -S netrumauto
   ```
   
5. Run Script
   ```
   ./netrum-auto.sh
   ```

CTRL A + D for run in background
